# Dockerfile Examples

This repository contains a set of examples of Dockerfiles for various technologies. Each folder contains a specific example demonstrating how to create a Dockerfile for a particular technology stack.

## Folder Structure

- `angular-app/`
  - Contains an example of a Dockerfile for an Angular application.
  - Includes a `README.md` file with instructions on how to develop, build, and test the Angular application.

- `html-app/`
  - Contains an example of a Dockerfile for a simple HTML application served by Nginx.
  - Includes a `Dockerfile` that sets up an Nginx server to serve static HTML, CSS, and JavaScript files.

- `nodejs-app/`
  - Contains an example of a Dockerfile for a Node.js application.
  - Includes a `.gitignore` file to exclude `node_modules` from version control.

## Usage

To use any of the Dockerfile examples, navigate to the respective folder and follow the instructions provided in the `README.md` file or the comments within the Dockerfile.

## Technologies Covered

- Angular
- HTML/CSS/JavaScript (served by Nginx)
- Node.js

## Additional Resources

For more information on Docker and Dockerfiles, visit the [Docker Documentation](https://docs.docker.com/).
